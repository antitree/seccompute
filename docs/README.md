# Documentation

This folder hosts project documentation and static assets.

- `assets/` contains static files used in docs and the repository README.
  - `images/` holds images such as the project logo.

If you plan to publish a site later:

- MkDocs: keep docs here and add a `mkdocs.yml` at the repo root.
- Sphinx: store sources here and use `docs/_static/` for assets.
